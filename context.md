## Secrets location
- .env at project root (ANTHROPIC_API_KEY, BRAVE_API_KEY)
- gcal_credentials.json at project root (download from Google Cloud Console)
  → APIs & Services → Credentials → OAuth 2.0 → Download JSON

## Do not modify these files directly
- memory_store.json    (only memory/store.py writes to this)
- gcal_token.json      (auto-regenerated by Google OAuth flow)
- logs/                (only agent/main.py writes here)

## When you add a new tool
1. Create tools/new_tool.py
2. Add it to tools/__init__.py
3. Add its config constants to agent/config.py
4. Update CLAUDE.md "File ownership map" section
5. Write a test in tests/test_new_tool.py
